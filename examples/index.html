YouTube iframe API sequence mashup

This page demonstrates how simple it is to embed a sequence of YouTube clips using the new Youtube iframe API, along with youtube-remix.js. The current version works best with Chrome.

Start by placing a div tag in your document. You can style this div tag however you want, to match the layout or positioning of your theme. Next, configure your mashup. Take a look at the source for this HTML file, and you'll see a variable called mash_cfg. This page's configuration will turn the div ID "example_player" into a youtube chromeless player, set its dimensions to 640x390, use a fading effect to transition between segments, and play a sequence of video segments. As the last step, it invokes youtube-remix.js which takes care of the rest!

The coolest part about the iframe API is that it is compatible with Flash and HTML5, so make sure you test this page twice. First, opt into the YouTube HTML5 trial, here. Then, leave the HTML5 trial and load this page a second time to test out the Flash version. The best part about youtube-remix.js is that you don't need to know any javascript at all to use it. Everything is controlled right here, in this HTML file! For another example, take a look at this.
